# AgentWatch

> AI Agent Security Monitoring Platform - 开发者平台安全基础设施

## 项目定位

- **核心功能**: Agent行为监控、供应链安全、开发者平台安全
- **差异化**: 中国市场优先 + 安全优先 + 开放架构
- **目标**: Year 1 ¥1016万/$145万 ARR

## 技术栈

- **Backend**: FastAPI + Python 3.11
- **Frontend**: React + TypeScript
- **Database**: ClickHouse (时序数据)
- **Infra**: Docker + Kubernetes
